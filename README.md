# IIOU - I Owe You App

An application for managing credits and debts based on the Pi Network.

## Description
IIOU helps users manage their personal loans, ensuring clear traceability and facilitating repayments through secure Pi payments.

## Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript with Pi SDK integration.
- **Backend**: Vercel Serverless Functions (Node.js).

## Configuration
1. Clone the repository.
2. Set the environment variables on Vercel:
   - `PI_API_KEY`: Your API key provided by the Pi Developer Portal.
3. Ensure that the API functions are located in the `/api` folder with a `.js` extension.

## Developer Info
- **App Network**: Pi Testnet
- **Hosting**: Vercel (Self-Hosted)
